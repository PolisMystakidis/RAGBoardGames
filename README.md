# RAG practice project
This is a project for practicing using RAG systems on english PDF files so I can finally use it on greek PDF files.
You need to install Ollama to run https://ollama.com
Once you download it run 
** ollama pull nomic-embed-text:latest **

for the embeddings model and 
** ollama pull deepseek-r1:1.5b **

for the text generator model.
